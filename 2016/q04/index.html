平成28年度 問4 有機化学及び燃料

アミンの反応や生成に関する次の反応（A）～（C）の名称の組み合わせとして、最も適切なものはどれか。
① Hofmann転移 / Sandmeyer反応 / Curtius転移
② Curtius転移 / Hofmann転移 / Sandmeyer反応
③ Curtius転移 / Sandmeyer反応 / Hofmann転移
④ Sandmeyer反応 / Hofmann転移 / Curtius転移
⑤ Sandmeyer反応 / Curtius転移 / Hofmann転移
正答
④ Sandmeyer反応 / Hofmann転移 / Curtius転移
Aはは芳香族アミンからジアゾニウム塩を経て、最終的に臭素化された芳香族化合物を生成しているためSandmeyer反応です。 Bはカルボン酸アミドからアミンを生成する反応で、二酸化炭素を副生成物として放出しているためHofmann転位です。 Cは酸塩化物からアジド中間体を経て、最終的にアミンを生成しています。二酸化炭素と窒素ガスを副生成物として放出する点が特徴的で、これはCurtius転位です。
